**Q: I can't login, what should I do?**
A: Ensure the backend server is running and the database is connected. Check the console for error messages regarding API connection.

**Q: The images are not loading.**
A: Check if the Cloudinary environment variable `VITE_CLOUDINARY_CLOUD_NAME` is correctly set in your `.env` file.

**Q: Socket connection fails.**
A: Verify that the `VITE_SOCKET_URL` points to the correct backend URL and port (default 5000).
